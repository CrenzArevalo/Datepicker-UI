# Datepicker UI - Roadmap.sh Project

This is a custom-built, interactive Datepicker user interface created as part of the [roadmap.sh](https://roadmap.sh/projects/datepicker-ui) frontend challenges.

## Features
- **Semantic HTML**: Uses `<table>` for the calendar grid and proper header tags.
- **CSS Positioning**: Implements `position: absolute` for the dropdown effect.
- **Zero JavaScript**: Built entirely with HTML and CSS.
- **SEO Optimized**: Includes meta tags for search engine visibility.
